GET config/pairs/:id
=======================================================

<em>Returns an endpoint pairing by its identifier.</em>

Entity Type
-----------
Pair

URL
---
http://server:port/diffa-agent/rest/config/pairs/:id

 
Mandatory Parameters
--------------------

### id

*string*

Pair ID

Requires Authentication
-----------------------
no
