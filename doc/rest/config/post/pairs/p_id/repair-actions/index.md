POST config/pairs/:id/repair-actions
=======================================================

<em>Creates a new repair action associated with a pair that is registered with the agent.</em>

Entity Type
-----------
Response

URL
---
http://server:port/diffa-agent/rest/config/pairs/:id/repair-actions

 
Mandatory Parameters
--------------------

### id

*string*

Pair ID
